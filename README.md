## <img src="https://cdn.jsdelivr.net/gh/devicons/devicon/icons/java/java-plain.svg" width="80" height="80"/>

#### Java
```java

    }
}
```

## <img src="https://cdn.jsdelivr.net/gh/devicons/devicon/icons/c/c-original.svg" width="80" height="80"/>

```c

int main() {
   printf("Hello World");
   return 0;
}
```

## C++ 

```cpp

}
```
